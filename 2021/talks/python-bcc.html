Python 与 BCC 的结合：内核可观测性的新手段

10/1615:30 - 16:15

李者璈太极图形资深工程师，微软 MVP

  • 猫奴
  • 一个热爱 Python 的人
  • 太极图形搬砖者，欢迎一起搬砖

随着云计算的进一步发展，我们所要面临的线上问题也越来越复杂化。我们可能在线上排查问题时需要去深入到内核中，去获取包含并不仅限于网络包，进程信息在内的各种相关的信息。这也让我们对 Linux 内核调试工具的需求也日渐增加

在 Linux 4.x 之后，Linux 在内核中提供了 eBPF 在内的良好的工具让我们能够升入到内核去获取到我们所需要的信息。

本次演讲将带大家从一些实际的例子出发，去一点点感受 Python + BCC 给我们线上调试所带来的便利
