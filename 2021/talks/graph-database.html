演讲分会场 A

图数据库解谜与上手应用

10/1713:00 - 13:45

古思为Vesoft Inc.

我是一个在上海的软件工程师，我在 vesoft 担任 Nebula Graph（开源的分布式图数据库）的社区开发者布道师。

我的工作是通过围绕 Nebula Graph Database 创作内容，构建工具来改善开发者的学习、开发、社区参与体验。

我在开源社区开放的工作，并（花了职业生涯中的前些年意识到）热爱用自己的思想和学到的技术帮助到别人，我认为这是一种的荣幸和宝贵的机遇。

  • 我的介绍更多内容： https://siwei.io/about/
  • 我的部分往期 talk： https://siwei.io/talk/
  • 个人 twitter： https://twitter.com/wey_gu

过往经历

我曾在爱立信工作了近十年：2011 年到 2021 年。 角色是云计算产品 Cloud Execution Envrioment (CEE) 研发团队的系统经理 ，我大部分的工作是通过设计开发 20 多个 CEE 6.6.2 和 CEE 10 的功能与改进，涵盖计算、存储、网络、生命周期管理和安全等领域，来帮助爱立信 IaaS 产品与解决方案不断进化。 同时，我也负责 CEE 产品在中国区的布道（面向外部与内部）。

《开源面对面》播客的主持人（可惜还只发出了第零期）

图数据库是什么？Why Yet Another Database? 它能解决什么问题？ 本次演讲会给大家解谜图数据库，并分享、Demo 从零到一，分享讲者自己用 Python 搭建的基于图数据库做出来的有趣开源项目：

  • Nebula Siwi，一个基于图数据库 + Python 搭建的全栈单一领域智能问答机器人
  • 股权穿透，一个基于 Python Faker 生成数据，演示股权关系穿透查询的小项目
