议程

上午
09:00 签到 & 开场
09:30 开场
09:45 How CODE works
讲义 		徐涛 介绍豆瓣的 CODE 的内部机制,一个 GitHub 的 Copy. CODE 是基于 Python 的应用,介绍一下具体的技术实现,与开发过程的一些经验.
10:25 Python 在知道创宇的应用--- 软件构建实践分享
讲义 		郑堃 Python 都用在了哪些类型的开发上;代码管理的实践;单元测试的实践;code review 的实践;包管理/构建发布的实践;持续集成的实践;等等软件构件中涉及的大小主题. 部分内容已经分享到了:http://pm.readthedocs.org/
11:05 一起来设计云计算平台的任务调度系统
讲义 		董伟 技术人擅长学习新技术,有好奇心,希望攻克技术难题. 那么想不想一起参与设计一次商业云平台的任务调度系统呢?如果你愿意,欢迎一起参与到本次的演讲中来. 我们将和现场所有的Pythoner一起从头完成一次商业云计算平台的任务调度系统的设计.
11:45 APM - Python Application Performance
讲义 		高驰涛 在互联网,大数据发展的今天,应用性能管理(APM)已经越来越成为研发与运维所关注的难点与痛点.此次着重面向开发者与运维人员,分享云智慧在APM产品中,怎样运用先进的SmartAgent技术处理应用性能管理,更快更全面地发展已有的或潜在的性能瓶颈.
中午
12:15 午餐+合照
下午
13:10 OpenStack 的软件架构
讲义 		王为 以 OpenStack 的虚拟网络组件 Neutron 为例介绍 OpenStack 的软件架构,兼谈健壮,可伸缩,低耦合的分布式系统设计.
13:50 current.futures 研究
讲义 		laike9m current.futures 是我非常喜欢使用的一个并发模块,相对 multiprocessing,它提供了更为简洁的接口. 希望能介绍这个模块的基本使用,并且深入源码探究它的实现.
14:30 花蓮.py 社区体验
讲义 录音 录像 		魏泽人 我的开源体验,如何在没有人用 Python 的偏乡,建立 Python 社区的一些小小经验. 这三四年参与开源社区活动,并尝试将从社区学到的经验,回馈到地方.
15:00 工作坊(专用教室进行)
讲义 		林帅 现场教大家用 Scrapy 框架做网络爬虫,以及面向开发者的网络爬虫 PaaS - ScrapyCloud. 具体内容包括从最开始怎么写爬虫,到用一个 SAE 上的应用来展示爬虫获取的数据. 最后如果时间允许可以再简单介绍一下不用写代码的自动爬虫工具 Auto-Scraping.

闪电演讲
15:20 QPython - 安卓上的Python
讲义 		曲池峰 介绍QPython,如何在手机和平板上写Python程序,在手机上用Python写出实用的脚本程序,随时随地写程序.
15:30 Python代码的多平台迁移
讲义 		陈剑(寒晨) 讲讲Python代码在多平台迁移时遇到的那些坑. 总结如何才能写出能够平滑进行多平台迁移的Python代码. 主要涉及硬编码,依赖导入,系统特性,发布方式等问题.

讲义 		刘晓明 Python在机器人技术中的优势与应用
16:00 Python 在智能硬件领域的尝试
讲义 		杜军 悦Pod 后台内容聚合分析部分采用了Nginx + Tornado + Flask 的结构,完美完成任务的同时极大的节省了团队资源;另外在悦Pod前期原型测试的时候,悦Pod 团队采用 Python + RaspBerry 的方式快速验证想法,功能成熟之后迅速切换到生产模式,非常方便. 总而言之,Python在悦Pod 从模型验证阶段一直到正式量产支持都提供了接近完美的支持,强烈推荐在智能硬件方面有想法的团队使用!人生苦短,我用Python!#插播广告,悦Pod致力于智能音箱领域的推广和普及,目标是加速互联网音频内容流动,作为互联网内容的出口,悦Pod提供7x24小时待机,让像水电的使用一样方便,悦Pod就是音频内容的水龙头,电灯开关.
16:10 pyspider 爬虫框架介绍
讲义 		binux 两种无需编写代码的结构化数据抽取(http://blog.binux.me/2014/07/how-to-extract-data-from-web/ 中介绍的 data highlighter 和 自动模板生成与提取)演示
16:20 产品开发在豆瓣
讲义 		蔡斌 通过讲述豆瓣东西的开发经历,介绍一下豆瓣工程师在一个产品开发过程中的工作方式
16:30 走进中国科技网
讲义 		谢建军 主要内容是科技网的网络接入情况,科研服务及软件开源情况的介绍.
16:40 如何参与开源项目
讲义 		梁杰 开源大家都听过,但是真正参与到开源项目中的程序员却少之又少. 在这个快速演讲中我会介绍如何参与开源项目,让每个程序员都能更快地得到提升.
