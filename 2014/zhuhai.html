议程

上午
09:30 签到
10:00 暖场 (大妈); 录音
10:15 使用Python实现 "Go元编程"
讲义 录音 		翁伟(Wuvist) 讲述如何使用python对go语言进行预处理生成代码,实现 "Go元编程";目前项目Go源码已经突破30W行,大半是由python"编译器"自动生成而来.
11:00 一切皆python的易度前端框架
讲义 录音 		潘俊勇 前端技术变化非常快,前端知识快速陈旧,javascript语言很蛋疼,html也很蛋疼,多终端支持也很蛋疼. 易度自己构建的前端框架,一切皆Python,快速编写前端应用,无js,无html,无蛋疼,支持多终端.
中午
11:50 (ー`´ー)
下午
13:00 stackless py 打造微内核的流程引擎
讲义 录音 		陶文 芯雲(bpm.py) 业务流程管理系统. 独特之处在于不需要学习新的流程定义语言. 如果你会写python代码,那么你在十分钟之内学会定义一个流程.甚至还支持多机并行执行,使用bpm.py做一个小的map/reduce分布式作业都是可能的!
13:40 keep simple things simple and high performant
讲义 录音 		黄毅 分享cython开发实践,以及两个python库cprotobuf和credis的设计.
14:30 花蓮.py 社区体验
讲义 录音 录像 		魏泽人 我的开源体验,如何在没有人用 Python 的偏乡,建立 Python 社区的一些小小经验. 这三四年参与开源社区活动,并尝试将从社区学到的经验,回馈到地方.
15:00 茶歇
15:10 软硬结合 For Pythoneer
讲义 录音 		王安宇 移动互联网下智能硬件是一个绕不开的话题,开源硬件给移动互联网带来了很大的想象空间,也让更多人可以参与移动互联网---尤其是程序员. 针对目前开源硬件的现状,流行的平台模块和应用案例给大家做分享交流,希望给各路饥渴的程序员一些新的思路和启发
15:50 Docker Registry Build By Python
讲义 录音 		孟繁亮 讨论在Docker 体系中 Python 的能力
16:00 茶歇
16:30 python是否适合高性能web服务的开发?
讲义 录音 		温铭 主要从异步,缓存等方面探讨.
17:00 Luna 之高效 KV 库
讲义 录音 		龙利民 在大数据实时统计系统架构过程中,必然的遇到了海量任务队列的分布式储存问题;分享如何一步步,用合适的开源系统升级架构,到最后,基于开源产品,完成原创系统的体验.

17:15 Python web异步编程实践
