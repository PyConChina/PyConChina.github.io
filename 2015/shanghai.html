13:30 用 jieba 处理中文以及台湾的 Py 社团
亮亮 首先分享在 Pinkoi 實習的經驗，如何使用 jieba 與改善 Elasticsearch 中文商品搜尋體驗，同時分享調整 jieba 斷詞於搜尋使用的一些經驗。再進一步透過 word2vec 自動找出商品間的關係，利用它語意上的特徵，完成一些如相似商品推薦等進階的搜尋功能。剩下的時間將會介紹現階段台灣 Python 社群發展以及宣傳明年活動。
14:00 Python应用的性能优化实践
刘国强 (Bruce Liu) 如何能够追踪和定位Python应用的程序代码性能？使用云智慧的应用性能平台能够快速定位和发现Python应用的性能瓶颈，并找到有问题的代码。使用云智慧的APM ，您可以深入到运行在实际生产环境中的Python应用程序，从交易级了解Python程序堆栈跟踪诊断性能瓶颈。可以自动检测商业交易，了解响应时间、每分钟调用时间，深入到代码级的诊断显示完整的调用图和执行代码，同时了解到Python调用SQL语句的执行性能。在本次演讲中将我们将分析PythonCode如何插入Python解释器，并采集Python代码堆栈、资源利用情况、代码的错误异常数据等，最终让开发人员和业务人员能够在最短时间内发现问题，降低对业务的影响。
14:30 使用Python进行高效大数据分析处理与可视化
丁来强 大数据处理中;70%左右的时间我们在做处理准备(加载/抽取/清洗/转换/分组/ 合并等); 进行商务分析的过程中(以及之后);又会花相当多的时间做数据可视化的工作. 这个主题介绍了如何使用Python选择高效的数据处理分析与可视化.内容结合一个实际例子覆盖了如何使用Pandas进行高效数据ETL; Pandas的最新特性(Query/ 去GIL等); BokeH如何进行高效数据可视化; Blaze与Spark结合进行更大规模的分布式数据处理等.
15:00 Python&GUI
陈海波 国内貌似对于Python&GUI上能直接看到的资料，基本停留于Hello World的阶段，没有梳理过，但内容应该会有下面这些： Why GUI 基本技术路线的选择 不同平台上的编译、打包软件 界面怎么绘制？ ... GUI赋予的想象力 如何开始你的第一个桌面软件？
15:30 茶歇
15:40 图形化python—快速构建机器人系统
王立奇 1.发布公司旗舰软件产品：Python机器人NAO基于NI LabVIEW的开发包。 2.介绍利用LabVIEW调用Python模块，并开发机器人应用程序的方法。 3.介绍利用VIRobotics软件产品将NI LabVIEW与Python交叉编译的方式。 4.介绍利用开发包添加Python模块的方式。
16:10 TheConsortium: 基于状态的数据中心自动化管理工具
徐洲 TheConsortium 是一个构建于 tornado, etcd 和 arangodb之上的数据中心管理和配置工具。基于 Consortium 的 plugin 系统，它可以与数据中心的各种后端，比如数据库，交换机，负载均衡器，puppet 交互，从而提供一个统一的入口来管理数据中心的各个组件。
16:45 一种远程执行代码的方法
许智翔(shell909090) 10分钟的小演讲，介绍一种将任何代码扔去远程执行的小技巧。
17:15 python中的数据工具箱
肖凯 介绍近年来python社区涌现的一些数据分析工具，包括numpy/scipy/pandas/sklearn/theano等等。还会介绍这些工具箱在数据分析工作流程中的位置，以及和其它类似工具的比较。
闪电演讲
17:30 py + gi - 高效 C 庫整合方案
Derek 呆 要從 Python 使用 C libraries, 通常需要在中間做上一層 binding, 讓 Python runtime 與 target C libraries 能互通, 維護費力易出錯. 經由導入 gobject introspection, 配合 type information, 讓 Python + C 的組合產能最大化!
17:40 使用 Luiti 来构建数据仓库
陈大伟 在数据量级上升到必须用 Hadoop 去处理的时候，一个易用且高效的离线任务管理框架必不可少。Luiti 正是这样从半年多实践中诞生的务实派开源作品，它是基于 Spotify 开源的 Luigi 模块之上构建起来的，可以像使用 Flask 和 Rails 等 Web 框架一样迅速搭建并运转起整个大型数据仓库。更多请访问 http://luiti.github.io/chinese/index.html
