Python 集市展

PyCon China 2024 集市展‍‍‍

PyCon China 2024 面向所有开发者社区，开放 Python 集市，汇聚众多 Python 领域的技术社区和企业在现场开设展位，与大家一同交流互动。此外，还有丰富有趣的游戏环节等你来参加，让你在轻松愉快的氛围中更好地体验 Python 的魅力；参与集市还将有机会赢得来自 PyChina 以及各企业、社区的周边好礼，这是一个让你不仅可以扩展 Python 知识和技能，还能赢得实用好礼的机会！

在这里，诚邀各个公司、社区、工作室或者个人来摆摊，欢迎大家一起参与 Python 爱好者的开源聚会！

  • 现场逛集市展区
  • 签到合影集赞
  • 打卡赢取精美周边
  • 更多趣味玩法等你解锁
  • ……

我们希望每一个参与者都可以收获满满！

🧡 您可以寻找到志同道合的战友，切磋交流技术；

🧡 您可以直面各行业开发者，畅聊昨日的 Bug；

🧡 您可以获得与潜在用户对话的机会，说不定他就是你下一个技术伙伴。

集市小贴士

本次参展免费，大会将为大家提供摊位桌椅。

您需要准备：

  • 活动前协助 PyCon China 2024 大会宣传；
  • 活动当天工作人员现场摆摊；
  • 参展需设置互动环节（鼓励大家多带周边小礼物与观众互动，多多展示自己的开源项目）；
